# Tax Expense Tracker Skills

These skill files break the project plan into focused execution guides.

## Skill Files

- `backend/SKILL.md`: API, EF Core, SQLite, soft-delete, CRUD endpoints
- `frontend/SKILL.md`: Angular app setup, service layer, component roadmap
- `azure-deployment/SKILL.md`: Azure App Service and Azure Static Web Apps deployment flow

## How to Use

1. Start with `backend/SKILL.md` to complete Phase 1.
2. Move to `frontend/SKILL.md` for Phase 2.
3. Use `azure-deployment/SKILL.md` during Phase 4 or earlier for environment prep.
